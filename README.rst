Mall Customers Segmentation using Unsupervised Machine Learning
===============================================================
Applies unsupervised learning techniques to identify meaningful customer segments within a dataset. 
The task requires comparison of two clustering approaches and a critical interpretation of the resulting clusters in an application context.

Clustering Algorithms Used
--------------------------

1. K-Mean
2. Agglomerative 

Build Steps
-----------

1. Open your RapidMiner application
2. Import the file **Customer Segmentation Using Unsupervised Learning.rmp** into RapidMiner
3. Import the **Mall Customers.xlsx** dataset via the Import Data wizard in RapidMiner
4. Now, select the **Retrieve** operator and set its repository entry to the imported **Mall Customers.xlsx** dataset
5. Finally click on **Run** button to execute the pipeline
6. Click the "Results" tab to view the results of the pipeline execution.
